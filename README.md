# Olive June Math - Portfolio CV

Portfolio cá nhân dạng CV trực tuyến của Olive June Math, sinh viên năm thứ nhất ngành Khoa học Máy tính và Thông Tin tại VNU-HUS.

## Website

Live demo: https://liviaisthebest911-alt.github.io

## Công nghệ

- HTML5 semantic
- CSS3 thuần, responsive, print stylesheet
- Vanilla JavaScript
- GitHub Pages

Không cần framework, package manager hoặc build tool.

## Tính năng chính

- Giao diện CV trực tuyến hiện đại, tối giản và có vibe developer portfolio.
- Header có terminal card tinh tế, không dùng các ký tự phân tách thủ công kiểu `//`.
- Stats, skills, projects, notes và contact được tổ chức thành card/grid rõ ràng.
- Nút Copy email nhanh.
- Nút Tải CV / In PDF dùng `window.print()`.
- `@media print` tối ưu riêng để Ctrl+P hoặc lưu PDF ra bản CV gọn, nền trắng, ẩn navigation và các hiệu ứng trang trí.
- Responsive tốt trên mobile và desktop.
